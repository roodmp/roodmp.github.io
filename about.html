How This Site Was Made

A brief overview of how this site was made and what tools were leveraged to make it work!

01

Recreated Resumé in HTML

Initially tried exporting my resumé from Notion but their minified HTML export caused issues when trying to adjust values so for manageability and maintenance sake, opted to recreate it from scratch.

02

Azure Static Website

Created a new storage account, set it for static website (vs. blob storage) and made note of the primary endpoint.

03

Custom Domain Name

I first transferred my custom domain name (MarkRood.io) from Google Domains to Cloudflare for a better management experience (IMO). Next using the primary endpoint noted in the previous step, I create a CNAME record in Cloudflare so that www.MarkRood.io would point to the Azure endpoint.

04

HTTPS

By leveraging Azure's CDN, I was able to create a CDN profile with a new secure endpoint. Then edited my Cloudflare DNS and replaced the old endpoint with the new CDN provided one.

05

CI/CD via GitHub Actions

Up to this point any changes to the code had to be manually uploaded to the Azure storage account. To make it easier to deploy changes, I configured the Github resume repo to use GitHub Actions and on every commit, it authenticates with Azure (RBAC), uploads the repo, purges CDN (of cached items) and finally logs out of Azure.

06

Tailwind CSS

With the infrastructure relatively where I wanted it, I decided to change gears and switch to design. This being my first time using Tailwind was happy to find out it's relatively similar to Bootstrap and how they handle the styling via classes. Would have loved to not reformat my code..again but c'est la vie. A good learning experience!
